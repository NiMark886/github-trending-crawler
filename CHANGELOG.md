All notable changes to this project will be documented in this file.

## [1.2.0] - 2026-05-27

### Added
- Logging system with colored output and file rotation
- Proxy manager with health checking and rotation
- SQLite database for data persistence
- Star history tracking
- Data visualization with matplotlib
- Rich library for beautiful terminal output
- GitHub Actions CI/CD pipeline
- Issue and PR templates
- CHANGELOG.md
- CONTRIBUTING.md

### Changed
- Improved error handling throughout the codebase
- Enhanced progress bar display
- Better configuration management

### Fixed
- Network timeout handling
- Unicode encoding issues on Windows

## [1.1.0] - 2026-05-27

### Added
- Multi-threaded concurrent downloads
- Repository search functionality
- Daily/weekly/monthly trending support
- Interactive selection with pagination
- Configuration management system
- Custom download directory selection

### Changed
- Improved CLI interface
- Better progress tracking
- Enhanced error messages

## [1.0.0] - 2026-05-27

### Added
- Initial release
- GitHub trending crawler
- Basic download functionality
- JSON export
- Rate limiting
- README documentation
- MIT License
