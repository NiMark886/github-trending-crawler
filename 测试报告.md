## 🚀 使用示例

### 示例 1: 搜索并选择下载

```bash
# 搜索 Vue 仓库并选择下载
python cli.py select search "vue" -n 20

# 交互式操作:
# 输入: 1,3,5  (选择第1、3、5个)
# 输入: done   (完成选择)
# 输入: y      (确认下载)
```

### 示例 2: 设置下载目录

```bash
# 查看当前配置
python cli.py config show

# 设置下载目录
python cli.py config set-dir ~/Desktop/my_projects

# 交互式选择目录
python cli.py config set-dir
```

### 示例 3: 热榜并选择下载

```bash
# 获取本周热榜并选择下载
python cli.py select trending

# 获取日榜并选择下载
python cli.py select trending -s daily

# 获取月榜并选择下载
python cli.py select trending -s monthly
```

### 示例 4: 指定下载目录

```bash
# 使用 -o 参数指定下载目录
python cli.py select search "react" -o ~/Desktop/react_projects

# 下载到指定目录
python cli.py download repos.json -o ~/Desktop/downloads
```

## 🎨 CLI 命令汇总

| 命令 | 说明 | 示例 |
|------|------|------|
| `crawl` | 获取热榜 | `python cli.py crawl --since daily` |

| `download` | 下载仓库 | `python cli.py download repos.json` |

## ⚙️ 配置选项

| 命令 | 说明 |
|------|------|
| `config show` | 显示当前配置 |
| `config set-dir <path>` | 设置下载目录 |
| `config set-dir` | 交互式选择下载目录 |
| `config set-workers <n>` | 设置并发线程数 |
| `config set-rate <seconds>` | 设置请求间隔 |
| `config reset` | 重置为默认配置 |

---

## 💡 提示

1. **下载目录**: 使用 `config set-dir` 设置默认下载目录
2. **并发线程**: 建议 3-10 个线程，过高可能被封
3. **请求间隔**: 建议 1-3 秒，避免频繁请求
4. **交互式选择**: 支持 `1,3,5`、`1-5`、`all` 等选择方式
5. **配置持久化**: 配置保存在 `~/.github_crawler/config.json`
