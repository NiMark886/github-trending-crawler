## 🎯 核心功能

### 1. 数据采集
- ✅ 获取 GitHub **日榜/周榜/月榜** 热门项目
- ✅ 支持按编程语言过滤
- ✅ 提取项目名称、描述、星标、Fork 数等信息
- ✅ 自动保存为 JSON 格式

### 2. 仓库搜索 (NEW!)
- ✅ 按关键词搜索 GitHub 仓库
- ✅ 显示作者名称、星标数、Fork 数
- ✅ 支持按语言、星标排序
- ✅ 结果导出为 JSON

### 3. 多线程下载
- ✅ 并发下载多个仓库
- ✅ 可配置线程数 (3-10)
- ✅ 进度条显示
- ✅ 断点续传支持

### 4. 数据分析
- ✅ 语言分布统计
- ✅ 星标排行榜
- ✅ 周增长趋势
- ✅ 数据可视化支持

### 5. 伦理爬虫
- ✅ 请求频率限制 (Rate Limiting)
- ✅ 浏览器 User-Agent 模拟
- ✅ 错误处理和重试机制
- ✅ 遵守 robots.txt 规范

## 🛠️ 技术栈

- **Python 3.7+**
- **requests** - HTTP 请求
- **BeautifulSoup4** - HTML 解析
- **lxml** - XML/HTML 解析器

## ⚠️ 重要注意事项

### 1. 法律合规性
- **GitHub 服务条款**: 自动化爬虫可能违反 ToS
- **robots.txt**: 必须遵守爬取规范
- **数据使用**: 仅限学习和研究用途

### 2. 技术风险
- **IP 封禁**: 频繁请求可能被封禁
- **页面变化**: GitHub 可能更新 HTML 结构
- **反爬虫**: 可能触发验证码或限制

### 3. 最佳实践
- ✅ 使用 GitHub API 替代爬虫
- ✅ 添加 1-3 秒随机延迟
- ✅ 设置合理的 User-Agent
- ✅ 实现完善的错误处理
- ✅ 记录详细的日志

## 🔄 替代方案

### 1. GitHub API (推荐)
```python
import requests

headers = {'Authorization': 'token YOUR_TOKEN'}
response = requests.get(
    'https://api.github.com/search/repositories',
    headers=headers,
    params={'q': 'stars:>1000', 'sort': 'stars'}
)
```

### 2. GitHub CLI
```bash
gh api "/search/repositories?q=stars:>1000&sort=stars"
```

### 3. 第三方服务
- ossinsight.io
- star-history.com
