## 🚀 快速开始

### 1. 安装依赖

```bash
pip install -r requirements.txt
```

### 2. 基本使用

```bash
# 获取本周热榜
python cli.py crawl

# 搜索仓库
python cli.py search "react"

# 交互式选择下载
python cli.py select search "vue"
```

### 3. 使用 Docker (可选)

```bash
# 构建镜像
docker build -t github-crawler .

# 运行
docker-compose up -d
```

## 📦 功能详解

### 热榜爬取

获取 GitHub 每日/每周/每月热门仓库。

#### 命令格式

```bash
python cli.py crawl [选项]
```

#### 选项说明

| 选项 | 说明 | 默认值 |
|------|------|--------|
| `-s, --since` | 时间范围 (daily/weekly/monthly) | weekly |
| `-l, --language` | 编程语言过滤 | 无 |
| `-n, --top` | 只显示前 N 个 | 全部 |
| `-o, --output` | 输出文件名 | 自动生成 |
| `-a, --analyze` | 显示趋势分析 | 否 |
| `-r, --rate-limit` | 请求间隔秒数 | 2.0 |

#### 使用示例

```bash
# 获取本周热榜
python cli.py crawl

# 获取今日热榜
python cli.py crawl --since daily

# 获取本月 Python 热榜前 20
python cli.py crawl --since monthly -l python -n 20

# 保存到指定文件
python cli.py crawl -o my_trending.json

# 显示趋势分析
python cli.py crawl -a
```

#### 输出示例

```

GitHub Trending Repositories
Retrieved: 2026-05-27 16:31:42
Total: 18 repositories
================================================================================

#1 colbymchenry/codegraph

   Language: TypeScript
   Stars: 28,802
   Weekly Growth: +0
   Forks: 1,624
   URL: https://github.com/colbymchenry/codegraph
--------------------------------------------------------------------------------
```

### 仓库搜索

按关键词、语言、主题搜索 GitHub 仓库。

#### 命令格式

```bash
python cli.py search <关键词> [选项]
```

#### 选项说明

| 选项 | 说明 | 默认值 |
|------|------|--------|
| `-n, --limit` | 最大结果数 | 20 |
| `-l, --language` | 编程语言过滤 | 无 |
| `-s, --sort` | 排序方式 (stars/forks/updated) | stars |
| `-o, --order` | 排序顺序 (asc/desc) | desc |
| `-f, --output` | 保存到 JSON 文件 | 无 |
| `-t, --token` | GitHub token | 无 |

#### 使用示例

```bash
# 搜索 react 仓库
python cli.py search "react"

# 搜索 Python 机器学习仓库
python cli.py search "machine learning" -l python -n 30

# 按 fork 数排序
python cli.py search "vue" -s forks

# 保存结果到文件
python cli.py search "angular" -o angular_repos.json
```

#### 输出示例

```
================================================================================
搜索结果 (5 个仓库)
================================================================================

   描述: The library for web and native user interfaces.
   语言: JavaScript
   ⭐ 星标: 245,287
   🍴 Fork: 51,115
   🔗 链接: https://github.com/facebook/react
--------------------------------------------------------------------------------
```

### 多线程下载

使用多线程并发下载仓库 ZIP 包。

#### 命令格式

```bash
python cli.py download <json文件> [选项]
```

#### 选项说明

| 选项 | 说明 | 默认值 |
|------|------|--------|
| `-o, --output` | 下载目录 | 使用配置 |
| `-w, --workers` | 并发线程数 | 5 |
| `-n, --top` | 只下载前 N 个 | 全部 |

#### 使用示例

```bash
# 下载 JSON 文件中的所有仓库
python cli.py download repos.json

# 指定下载目录和线程数
python cli.py download repos.json -o ~/Desktop/repos -w 8

# 只下载前 10 个
python cli.py download repos.json -n 10
```

#### 一键爬取+下载

```bash
# 爬取热榜并下载前 10 个
python cli.py all

# 指定参数
python cli.py all -s daily -l python -n 20 -w 8 -o ./python_repos
```

### 交互式选择

交互式浏览和选择要下载的仓库，支持分页和多选。

#### 命令格式

```bash
python cli.py select <来源> [关键词] [选项]
```

#### 来源类型

| 来源 | 说明 |
|------|------|
| `search` | 搜索仓库 (需要关键词) |
| `trending` | 热榜仓库 |

#### 选项说明

| 选项 | 说明 | 默认值 |
|------|------|--------|
| `-s, --since` | 时间范围 (仅 trending) | weekly |
| `-l, --language` | 语言过滤 | 无 |
| `-n, --limit` | 最大结果数 | 30 |
| `-o, --output` | 下载目录 | 使用配置 |
| `-w, --workers` | 并发线程数 | 5 |

#### 使用示例

```bash
# 搜索并选择下载
python cli.py select search "react"

# 热榜并选择下载
python cli.py select trending

# 日榜并选择下载
python cli.py select trending -s daily

# 指定语言和数量
python cli.py select search "vue" -l javascript -n 20
```

#### 交互操作

| `p` / `prev` | 上一页 | 翻到上一页 |

### 配置管理

管理默认下载目录、并发线程数等配置。

#### 命令格式

```bash
python cli.py config <操作> [值]
```

#### 操作说明

| 操作 | 说明 | 示例 |
|------|------|------|
| `show` | 显示当前配置 | `config show` |
| `set-dir` | 设置下载目录 | `config set-dir ~/Desktop/repos` |
| `set-workers` | 设置并发线程数 | `config set-workers 8` |
| `set-rate` | 设置请求间隔 | `config set-rate 3` |
| `reset` | 重置为默认配置 | `config reset` |

#### 使用示例

```bash
# 查看当前配置
python cli.py config show

# 设置下载目录
python cli.py config set-dir ~/Desktop/my_repos

# 交互式选择下载目录
python cli.py config set-dir

# 设置并发线程数
python cli.py config set-workers 8

# 设置请求间隔
python cli.py config set-rate 3

# 重置配置
python cli.py config reset
```

#### 配置文件位置

```
~/.github_crawler/config.json
```

#### 日志级别

| 级别 | 说明 |

| INFO | 一般信息 |
| WARNING | 警告信息 |

from logger import setup_logger, get_logger

# 初始化日志

### 代理支持

使用代理避免 IP 被封禁。

#### 命令格式

```python
from proxy import ProxyManager

# 初始化代理管理器
pm = ProxyManager(proxy_file="proxies.txt")

# 获取代理
proxy = pm.get_proxy()

# 使用代理请求
response = requests.get(url, proxies=proxy)

# 报告结果
pm.report_success(proxy['http'], response_time=0.5)
pm.report_failure(proxy['http'])
```

#### 代理文件格式

```
# proxies.txt
http://proxy1:8080
http://proxy2:8080
http://user:pass@proxy3:8080
```

#### 代理轮换策略

- **自动轮换**: 轮询使用代理
- **健康检查**: 自动检测代理可用性
- **失败重试**: 失败自动切换代理

### 数据库存储

使用 SQLite 存储历史数据，支持趋势追踪。

#### 使用方式

```python
from database import Database

# 初始化数据库
db = Database()

# 保存仓库
db.save_repos(repositories)

# 保存星标历史
db.save_star_count("facebook/react", 245000)

# 获取历史
history = db.get_star_history("facebook/react", days=30)

# 获取语言统计
stats = db.get_language_stats()

# 导出数据
db.export_to_json("export.json")
```

#### 数据库表

| 表名 | 说明 |
|------|------|
| repositories | 仓库信息 |
| star_history | 星标历史 |
| trending_history | 热榜历史 |
| search_history | 搜索历史 |

#### 数据库文件

- 位置: `github_trending.db`
- 格式: SQLite

### Docker 部署

使用 Docker 简化部署和运行。

#### 构建镜像

```bash
# 构建 Docker 镜像
docker build -t github-crawler .
```

#### 运行容器

```bash
# 运行爬取
docker run -v ./data:/app/data github-crawler crawl

# 运行搜索
docker run -v ./data:/app/data github-crawler search "react"

# 运行下载
docker run -v ./data:/app/data -v ./downloads:/app/downloads github-crawler download data/repos.json
```

#### 使用 Docker Compose

```bash
# 启动所有服务
docker-compose up -d

# 查看日志
docker-compose logs -f

# 停止服务
docker-compose down
```

#### Docker Compose 服务

| 服务 | 说明 |
|------|------|
| crawler | 爬虫主服务 |
| scheduler | 定时爬取服务 |

#### 数据持久化

```yaml
volumes:
  - ./data:/app/data          # 数据文件
  - ./downloads:/app/downloads # 下载文件
  - ./logs:/app/logs          # 日志文件
  - ./charts:/app/charts      # 图表文件
```

## ❓ 常见问题

### Q1: 下载速度慢怎么办？

**解决方案：**

```bash
# 增加并发线程数
python cli.py config set-workers 8

# 或者使用 -w 参数
python cli.py download repos.json -w 10
```

**建议线程数：**
- 国内普通网络：3-5
- 国内高速网络：5-10
- 海外网络：8-15

### Q2: 如何修改默认下载目录？

**方法 1：使用命令**

```bash
python cli.py config set-dir ~/Desktop/my_repos
```

**方法 2：交互式选择**

```bash
python cli.py config set-dir
# 然后选择常用目录或输入自定义路径
```

**方法 3：下载时指定**

```bash
python cli.py download repos.json -o ~/Desktop/specific_folder
```

### Q3: 搜索不到想要的仓库？

**尝试：**
1. 使用英文关键词
2. 增加结果数量：`-n 50`
3. 尝试不同的搜索词
4. 使用更通用的关键词

```bash
# 使用更通用的关键词
python cli.py search "web framework" -n 30

# 按语言过滤
python cli.py search "framework" -l python
```

### Q4: 下载失败怎么办？

**可能原因：**
1. 网络超时
2. GitHub 限制
3. 仓库不存在

**解决方案：**

```bash
# 使用代理
python cli.py config set-proxy http://proxy:8080

# 增加超时时间
python cli.py download repos.json -w 3

# 重试下载
python cli.py download repos.json --retry 3
```

### Q5: 如何查看日志？

**日志文件位置：**

```
logs/github_crawler_YYYYMMDD.log
```

**查看日志：**

```bash
# 查看今天的日志
cat logs/github_crawler_$(date +%Y%m%d).log

# 实时查看日志
tail -f logs/github_crawler_$(date +%Y%m%d).log
```

### Q6: 如何使用代理？

**创建代理文件：**

```bash
# 创建 proxies.txt
echo "http://proxy1:8080" > proxies.txt
echo "http://proxy2:8080" >> proxies.txt
```

**使用代理：**

```python

pm = ProxyManager(proxy_file="proxies.txt")
proxy = pm.get_proxy()
```

### Q7: 数据库文件在哪里？

**位置：** `github_trending.db`

**查看数据：**

repos = db.get_repos()
