## 五、技术建议

为降低使用风险，制作者建议使用者：

### 5.1 遵守爬虫规范

```python
# ✓ 推荐做法
- 设置合理的请求间隔（1-3秒）
- 使用浏览器 User-Agent
- 遵守 robots.txt 规范
- 控制并发请求数量

# ✗ 应避免的做法
- 频繁快速请求
- 无视 robots.txt
- 过度消耗服务器资源
- 绕过访问限制
```

### 5.2 优先使用官方 API

```bash
# 推荐使用 GitHub 官方 API
gh api "/search/repositories?q=stars:>1000&sort=stars"

# 或使用 Python
import requests
response = requests.get("https://api.github.com/search/repositories", ...)
```

### 5.3 控制使用规模

- 仅采集必要的数据
- 合理控制采集频率
- 避免大规模批量采集
- 尊重目标网站的资源
